# Edith Finch Asset Migration List

Assets to migrate from `D:\UnrealEngine Projects\EdithFinch` into Bastion Last Stand.
Open Edith Finch in a second UE5 editor instance, select assets, right-click > Asset Actions > Migrate, and point to `BastionLastStand\Content`.

After migration, move assets into `Content/Bastion/` subfolders and run Fix Up Redirectors.

## Migration Notes

- All source paths are relative to `EdithFinch/Content/`
- Migration will pull material and texture dependencies automatically
- After migration, reorganize into `Content/Bastion/Meshes/` subfolders:
  - `Weapons/`, `Tools/`, `Workbench/`, `Barricades/`, `Doors/`
  - `Containers/`, `Nature/Rocks/`, `Nature/Trees/`, `Structure/`
  - `Props/Survival/`, `Props/Furniture/`
- Run Fix Up Redirectors after moving assets
- Total estimated assets: ~150+ meshes (with materials and textures)
